This is Browser Engines GAnarchy. Currently tracking the following projects:

  • Julius Speech: Open-Source Large Vocabulary Continuous Speech Recognition Engine
  • NetSurf: Small & modular browser engine written in C
  • Stylish Haskell: Generic implementation of CSS selectors, specificity, cascade, and inheritance independant of supported CSS properties.
  • Vado: A demo web browser engine written in Haskell
  • eSpeak NG: An open source speech synthesizer that supports 107 languages and accents
